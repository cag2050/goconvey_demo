### 注意点：
1. 写测试文件，必须以`_test.go`结尾
2. 终端运行`$ $GOPATH/bin/goconvey`启动
3. 在浏览器打开：http://localhost:8080
4. 修改测试文件后，浏览器会自动刷新
5. 其他运行测试方法：1.ide中点击运行；2.终端运行`$ go test`或`$ go test -v`
